Linked Data Checker

This is proof of concept software that will check any JSON-LD embedded in web pages for conformance against a chosen schema. This instance is configured to check against a profile of schema.org and oerschema such as might be used to describe learning materials.

To try this out and compare the results to other similar testing services, have a look at the demo file or enter a URL here for checking:

Once again, this is proof of concept software. It is provided to illustrate what the checker may do. It is essentially pre-alpha in the development cycle; it changes frequently, and may or may not work on any given day. Do not use in a production environment. If you do want to make use of the software, contact the author via github.
