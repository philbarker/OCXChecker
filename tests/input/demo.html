Demo of Customizable LD checker

Embedded in this HTML is some JSON-LD metadata that uses oerschema and schema.org terms that could describe a lesson. Validating it in well known linked data checkers will raise some spurious errors and miss some real errors.

Google structured data testing tool

Perhaps the most widely used testing tool for structured data based on schema.org (the most widely used vocabulary).

Notes:

  • SDTT will complain about not knowing about terms from the oer: namespace
  • SDTT will complain about absence of terms that Google needs for some of its services
